# Lógica de Programação em Portugol e C

Este repositório contém meus exercícios de lógica de programação, para fixar o aprendizado, todos os exercícios resolvidos em portugol (visualg) foram portados por mim para a linguagem c.

Os exercícios em C não estão compilados, então caso queira executar os arquivos, terá de compilá-los.
